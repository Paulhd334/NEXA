Politique de Confidentialité

Nous protégeons vos données personnelles conformément au Règlement Général sur la Protection des Données (RGPD)

Phase de développement : Le système de compte est actuellement en phase de développement. Les comptes créés maintenant seront utilisés pour accéder au site web et, plus tard, seront synchronisés avec le jeu NEXA lors de son lancement.

Responsable du traitement

UNWARE STUDIO est responsable du traitement des données personnelles collectées via le site internet NEXA et, à terme, via le jeu NEXA.

Email : contact.unwarestudio@gmail.com

Siège social : Paris, France

Données collectées

Dans le cadre du site web NEXA et du futur jeu, les données suivantes sont collectées :

Données de compte

  • Identifiant Google unique
  • Adresse email Google
  • Nom d'affichage Google
  • Photo de profil Google

Données de site web

  • Préférences du compte
  • Date d'inscription
  • Dernière connexion

Finalités du traitement

Vos données sont traitées pour les finalités suivantes :

  • Gestion de votre compte via Google OAuth
  • Accès au site web NEXA
  • Synchronisation future avec le jeu NEXA
  • Support technique et client
  • Communication sur les mises à jour
  • Amélioration de l'expérience utilisateur

Consentement explicite

En créant un compte via Google, vous consentez expressément à ce que NEXA collecte et traite vos données personnelles (ID unique, nom, adresse e-mail, photo de profil) afin de :

  • Gérer votre compte utilisateur
  • Permettre l'accès au site web NEXA
  • Synchroniser vos préférences avec le jeu NEXA
  • Vous fournir les services demandés

Ce consentement est libre, spécifique, éclairé et univoque. Vous pouvez le retirer à tout moment en supprimant votre compte.

Bases légales

Le traitement de vos données repose sur :

  • Exécution du contrat : Nécessaire pour fournir l'accès au site web
  • Consentement : Pour les communications marketing
  • Intérêt légitime : Pour la sécurité et l'amélioration du service

Pour exercer vos droits, contactez-nous à l'adresse : contact.unwarestudio@gmail.com

Conservation des données

Vos données sont conservées :

  • Données de compte : 3 ans après la dernière activité
  • Données de site web : 2 ans après la dernière connexion

Sécurité des données

Nous mettons en œuvre des mesures techniques rigoureuses pour protéger vos données :

  • Authentification Google OAuth sécurisée : Élimination des mots de passe locaux et des risques associés
  • Stockage sécurisé : Utilisation de Supabase avec chiffrement des données
  • Accès restreint : Seul le développeur principal a accès aux données
  • Protocoles de sécurité : Mise en œuvre des meilleures pratiques de sécurité

Authentification Google

Pour simplifier et sécuriser votre expérience, NEXA utilise exclusivement l'authentification Google :

  • Pas de création de mot de passe : Évite les risques de fuite de mots de passe
  • Connexion simplifiée : Accédez rapidement à votre compte
  • Sécurité Google : Bénéficiez des protections de sécurité de Google
  • Récupération de compte : Via les mécanismes de sécurité Google

Important : Nous ne stockons jamais vos informations d'identification Google. Seul un identifiant unique est conservé pour associer votre compte Google à votre profil NEXA.

Transfert de données hors EEE

Les données collectées via Google peuvent être traitées en dehors de l'Espace Économique Européen (EEE).

Google met en place des garanties conformes au RGPD pour la protection de vos données, notamment :

  • Clauses contractuelles types de la Commission européenne
  • Certifications de conformité aux normes de protection
  • Mesures de sécurité techniques et organisationnelles

Synchronisation avec le jeu NEXA

Votre compte créé sur le site web sera utilisé pour :

  • Accéder au jeu NEXA lors de son lancement
  • Synchroniser vos préférences entre le site et le jeu
  • Maintenir un profil cohérent sur toutes les plateformes
  • Sauvegarder votre progression dans le jeu

Les données de jeu (progression, statistiques, etc.) seront collectées et traitées séparément lors du lancement du jeu, conformément à une politique de confidentialité mise à jour.
