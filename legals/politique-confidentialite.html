Politique de Confidentialité

Nous protégeons vos données personnelles conformément au Règlement Général sur la Protection des Données (RGPD)

Note importante : Le système de compte NEXA est actuellement fictif et en phase de développement. Aucune donnée personnelle réelle n'est collectée ou traitée pour le moment.

Responsable du traitement

UNWARE STUDIO est responsable du traitement des données personnelles collectées via le jeu NEXA.

Email : contact.unwarestudio@gmail.com

Siège social : Paris, France

Données collectées

Dans le cadre final du jeu NEXA, les données suivantes pourraient être collectées :

Données de compte

  • Nom d'utilisateur
  • Adresse email
  • Mot de passe chiffré

Données de jeu

  • Progression
  • Statistiques
  • Préférences

Données techniques

  • Type d'appareil
  • Logs techniques

Finalités du traitement

Vos données sont traitées pour les finalités suivantes :

  • Gestion de votre compte joueur
  • Sauvegarde de votre progression
  • Amélioration de l'expérience de jeu
  • Support technique et client
  • Communication sur les mises à jour
  • Prévention de la fraude

Bases légales

Le traitement de vos données repose sur :

  • Exécution du contrat : Nécessaire pour fournir le service de jeu
  • Consentement : Pour les communications marketing
  • Intérêt légitime : Pour la sécurité et l'amélioration du service

Conservation des données

Vos données sont conservées :

  • Données de compte : 3 ans après la dernière activité
  • Données de jeu : 5 ans pour les statistiques agrégées
  • Logs techniques : 1 an maximum
  • Données de facturation : 10 ans (obligation légale)

Cookies et traceurs

Nous utilisons différents types de cookies :

  • Cookies essentiels : Nécessaires au fonctionnement
  • Cookies de performance : Analyse d'audience
  • Cookies de fonctionnalité : Mémorisation des préférences

Vous pouvez gérer vos préférences cookies via les paramètres de votre navigateur.

Sécurité des données

Nous mettons en œuvre des mesures techniques et organisationnelles pour protéger vos données :

  • Chiffrement des données sensibles
  • Sauvegardes régulières
  • Accès restreint au personnel
  • Audit de sécurité régulier

Contact DPO

Pour toute question relative à la protection de vos données, contactez notre Délégué à la Protection des Données :

contact.unwarestudio@gmail.com

Délai de réponse : 30 jours maximum
