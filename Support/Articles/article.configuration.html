Configuration Système Requise pour NEXA

Dernière mise à jour: 15 janvier 2025
2,847 vues
5 min de lecture

📋 Vue d'ensemble des spécifications

NEXA est développé avec Unreal Engine 5, exploitant les dernières technologies graphiques comme Nanite et Lumen. Voici les configurations recommandées pour une expérience optimale.

Note importante

Ces spécifications sont basées sur la version de développement actuelle et peuvent évoluer avant la sortie finale.

💻 Configuration minimale

Permet de faire tourner NEXA en basse résolution avec des performances stables à 30 FPS.

Composant Spécification Notes
Processeur Intel Core i5-8400 / AMD Ryzen 5 2600 6 cœurs minimum
Carte graphique NVIDIA GTX 1060 6GB / AMD RX 580 8GB DirectX 12 requis
Mémoire vive 16 GB RAM Dual-channel recommandé
Stockage 80 GB SSD NVMe recommandé
Système Windows 10 64-bit Version 1909 ou supérieure

🚀 Configuration recommandée

Pour une expérience fluide en 1080p/60 FPS avec les paramètres graphiques élevés.

Composant Spécification Performance
Processeur Intel Core i7-10700K / AMD Ryzen 7 3700X 1080p @ 60 FPS
Carte graphique NVIDIA RTX 3060 Ti / AMD RX 6700 XT Ray Tracing basique
Mémoire vive 32 GB RAM DDR4 3200MHz
Stockage 80 GB NVMe SSD Chargements rapides
Système Windows 11 64-bit Optimisations récentes

🔥 Configuration Ultra

Pour profiter de NEXA en 4K/60 FPS avec tous les effets activés, y compris le Ray Tracing avancé.

Composant Spécification Performance
Processeur Intel Core i9-12900K / AMD Ryzen 9 5900X 4K @ 60 FPS
Carte graphique NVIDIA RTX 4080 / AMD RX 7900 XTX Ray Tracing complet
Mémoire vive 32 GB DDR5 6000MHz recommandé
Stockage 80 GB NVMe Gen4 Chargements instantanés
Système Windows 11 64-bit Dernières optimisations

⚙️ Fonctionnalités avancées

Ray Tracing

NEXA supporte le Ray Tracing pour des éclairages et reflets réalistes. Requiert une carte graphique RTX série 20/30/40 ou RX 6000/7000.

DLSS/FSR

Support de NVIDIA DLSS 3 et AMD FSR 2.2 pour améliorer les performances sans sacrifier la qualité visuelle.

Conseil d'optimisation

Activez DLSS/FSR si disponible pour gagner jusqu'à 40% de performances sans perte de qualité notable.

❓ Questions fréquentes

Puis-je jouer avec une configuration inférieure au minimum ?

Non recommandé. Le jeu pourrait ne pas démarrer ou avoir des performances inutilisables.

Les spécifications changeront-elles à la sortie ?

Des optimisations sont prévues, mais les configurations resteront similaires.

Support Mac/Linux ?

Seulement Windows pour le moment. Les autres plateformes seront évaluées après la sortie.
